# 开发
  ## 启动项目
  ```
  npm install
  ```
  首先启动react,其次启动electron
  ```
  npm run dev
  npm run start
  ```
# 打包
  ## 打包代码
  ```
  npm run prod
  ```
  ## 直接打包平台包
  ```
  npm run pack
  ```
  ## 打包压缩包(比较慢)
  ```
  npm run dist
  ```
